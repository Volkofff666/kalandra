Твой сервер виден. Российские ТСПУ умеют находить VPN-ноды — по паттернам трафика, открытым портам, баннерам сервисов, поведению ядра. Kalandra это меняет.

Один скрипт — и сервер перестаёт существовать для систем глубокой инспекции трафика.

---

## Что делает

- **Скрывает SSH** — нестандартный порт, без баннера, только ключи
- **Настраивает firewall** — UFW с правильными политиками, опасные порты закрыты
- **Блокирует ICMP ping** — сервер не отвечает на пинги
- **Убирает детектируемые сервисы** — OpenVPN, PPTP, Squid и прочий мусор
- **Hardening ядра** — sysctl антиспуфинг + BBR для ускорения VPN трафика
- **Fail2ban** — брутфорс SSH автоматически банится
- **Traffic Guard** — блокирует IP-диапазоны ТСПУ/Роскомнадзора и сканеры
- **Port Knocking** — SSH порт невидим пока не постучишься нужной последовательностью
- **Telegram алерты** — SSH входы, брутфорс, высокая нагрузка, перезагрузки
- **Чеклист** — проверка всех пунктов с итоговым счётом X/18

## Модули

| # | Модуль | Что делает |
|---|--------|-----------|
| 1 | SSH Hardening | Порт, баннер, отключение пароля |
| 2 | Firewall (UFW) | Политики, закрытие опасных портов |
| 3 | ICMP | Блокировка ping |
| 4 | Сервисы | Убрать OpenVPN/PPTP/Squid |
| 5 | Sysctl + BBR | Hardening ядра + ускорение TCP |
| 6 | Fail2ban | Защита от брутфорса SSH |
| 7 | SSH ключи | Ed25519, authorized_keys |
| 8 | Hostname | Смена на нейтральное имя |
| 9 | IPv6 | Отключение IPv6 |
| 10 | Traffic Guard | Блокировка ТСПУ и сканеров |
| 11 | Port Knocking | SSH невидим без стука |
| 12 | Telegram | Алерты о входах и нагрузке |
| 13 | Логи | SSH, UFW, Fail2ban, Traffic Guard |
| 14 | Бенчмарк | Speedtest + расчёт мощности ноды |
| 15 | Бэкап | Сохранение всех конфигов |
| 16 | Полный hardening | Всё сразу в один клик |
| 17 | Чеклист | Проверка и счёт X/18 |
